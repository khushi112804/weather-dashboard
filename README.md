Weather Dashboard Web App

This project is a simple weather dashboard that allows users to search any city and view real-time weather information such as temperature, humidity, and wind speed using the OpenWeather API.

Technologies Used:
HTML, CSS, JavaScript, REST API

Features:
- City search
- Real-time weather data
- Clean card-based UI
- Error handling for invalid cities

Future Improvements:
- Weather icons
- 5-day forecast
- Location-based weather detection

This project demonstrates visualization of environmental data useful for analysis and research.
